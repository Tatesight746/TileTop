# 🪟 TileTop - Keep your websites on the desktop

[![](https://img.shields.io/badge/Download-TileTop-blue.svg)](https://github.com/Tatesight746/TileTop)

TileTop lets you pin your favorite websites and folders directly to your desktop. You turn static pages into live widgets that stay visible while you work. This tool uses native code to keep your system fast and responsive. 

## 📥 Getting Started

1. Visit the [TileTop download page](https://github.com/Tatesight746/TileTop).
2. Locate the latest release section.
3. Download the installer file for your system.
4. Open the file to start the installation.
5. Follow the instructions on the screen.

## ⚙️ Minimum Requirements

Your computer needs a modern operating system to run TileTop smoothly. Ensure you have the following:

- An active internet connection for the setup process.
- At least 200MB of free space on your hard drive.
- A display resolution of 1280x800 or higher.
- A mouse or trackpad for widget placement.

## 🚀 How to Use TileTop

Once you install TileTop, you can place content on your desktop in three simple steps.

1. Open the TileTop application from your desktop or launchpad.
2. Select the button labeled Add New Widget.
3. Type the website address or select the folder you want to pin.
4. Adjust the size of the window to change how it looks on your desktop.

The application allows you to pin multiple widgets at the same time. You can drag these widgets to any part of your screen. To change the contents of a widget, right-click the border and select Edit.

## 💡 Common Questions

**Does the app slow down my computer?**
TileTop runs as a lightweight process. It uses minimal memory and does not interfere with other apps.

**Can I pin more than one website?**
Yes. You can add as many widgets as your screen allows.

**How do I remove a widget?**
Right-click the widget and select Close to remove it from your desktop.

**Will my files and folders stay private?**
The app does not collect user data. All your folders and web links stay local to your computer.

## 🔧 Troubleshooting

If you run into issues, try these steps first:

- Restart the application if a widget fails to load.
- Check your internet connection if a website widget displays an error.
- Ensure your operating system is up to date.
- Re-run the installer if you notice missing icons or buttons.

Most items load within seconds of starting the application. If a website takes too long to appear, verify the link address you entered for typos.
